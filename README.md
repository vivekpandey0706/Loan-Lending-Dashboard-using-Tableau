# Loan-Lending-Dashboard-using-Tableau
An interactive Tableau dashboard analyzing loan lending metrics, built from cleaned raw Excel data to track portfolio health, funding volumes,  and regional demand.

## Features
* **Dynamic Time Filtering:** A "Select Analysis Period" toggle filters all metrics by specific date ranges.
* **Granular Auditing:** A detailed bottom-pane ledger displays the top 100 loans, breaking down individual grades, purposes, terms, and amounts.
* **Performance Tracking:** Visualizes Month-over-Month (MoM) funding growth and tracks the overall ratio of good to bad loans.
* **Trend Analysis:** Categorizes applications by purpose (highlighting Debt Consolidation) and maps regional demand via state heat maps.

## How to View
* **Interactive:** Download `Loan Lending Dashboard.twbx` and open it with Tableau Desktop or the free Tableau Reader.
* **Static Snapshot:** View the included `Loan Lending Dashboard.pdf` for a quick, high-level overview.
